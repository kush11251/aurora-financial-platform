# Aurora Financial Platform
A robust financial analysis and portfolio management system built with Python and FastAPI.
## Requirements
- Python 3.9+
- fastapi
- uvicorn
## Installation
1. Clone the repository: `git clone https://github.com/username/aurora-financial-platform.git`
2. Navigate to the repository: `cd aurora-financial-platform`
3. Install dependencies: `pip install -r requirements.txt`
## Running the Application
1. Run the application: `uvicorn src.main:app --host 0.0.0.0 --port 8000`
